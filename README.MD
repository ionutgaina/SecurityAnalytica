# **SecurityAnalytica**

## **Start Backend**

1. you need to have docker installed on WSL or LINUX
2. go to rest_api folder
3. open here a command line and type

```
make
```

4. done

## **Start frontend**

1. install node js
2. go to frontend folder
3. open here a command line and type

```
npm i
npm run dev
```
4. done
